== Description ==

This plugin simply adds a report tab to display sales report by WP-Lister WooCommerce Reports. The plugin adds an additional report tab which display sales report by WP-Lister. You’ll find this report available in WooCommerce reports section.


== Installation ==

1. Upload the folder 'woo-sales-report-for-wp-lister` to the `/wp-content/plugins/` folder
2. Activate the plugin through the 'Plugins' menu in WordPress
